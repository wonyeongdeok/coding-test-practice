### Problem

### Lessons Learned

- MOD(x, y)
    - 나눗셈의 나머지 값을 반환하는 함수
    - x: 나눠지는 값
    - y: 나누는 값
- x DIV y
    - 나눗셈의 몫을 구하는 연산자
    - x: 나눠지는 값
    - y: 나누는 값
